## PI Monte-Carlo (CPU и GPU)

# Выполнил: Васильев Артём
# Курс: Высокопроизводительные вычисления, Самарский университет


В этой папке находится готовое к запуску решение задания из `pi_monte_carlo.pdf`.

### Что реализовано
- **Входные данные**: `N` — число случайных точек.
- **CPU**: оценка числа \( \pi \) методом Монте-Карло с помощью **NumPy**.
- **GPU (опционально)**: та же оценка через **CuPy** (CUDA), если доступно.
- **Выходные данные**: время выполнения и вычисленное значение **\( \pi \)** для CPU и GPU (значения могут отличаться, как указано в условии).

### Файлы
- `pi_monte_carlo.pdf`: условие задания.
- `HPC_PICalc_6131.ipynb`: блокнот с решением.

### Как запустить
1. Откройте `HPC_PICalc_6131.ipynb` в Jupyter / VS Code.
2. Выполните ячейки сверху вниз.
3. При необходимости измените входные параметры в ячейке **Входные данные**.

### Требования
#### CPU
- Python 3
- `numpy`

Установка:

```bash
python -m pip install numpy
```

#### GPU (опционально)
Нужны корректно установленные CUDA и версия CuPy, совместимая с вашей версией CUDA.

Установка (пример, пакет зависит от версии CUDA):

```bash
python -m pip install cupy
```

Если CuPy/CUDA недоступны, блокнот автоматически **пропустит запуск на GPU** и всё равно выдаст корректный результат для CPU.
