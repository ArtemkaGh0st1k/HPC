## Структура репозитория

Репозиторий содержит 5 лабораторных работ по параллельным вычислениям, каждая с реализацией на CPU и GPU:

### 1. **VecSum** — Сумма элементов вектора
- **Файл**: `HPC_VecSum_6131.ipynb`
- **Задача**: Параллельное вычисление суммы элементов вектора
- **Особенности**:
  - CPU-реализация: прямой проход по массиву
  - GPU-реализация (CUDA): редукция по блокам с использованием shared memory
  - Размер тестовых данных: N от 1 000 до 1 000 000 элементов
  - Ускорение GPU: ~47× на больших векторах
- **Выход**: Сравнение времени выполнения, проверка корректности

### 2. **Genetic** — Генетический алгоритм
- **Файл**: `HPC_Genetic_6131.ipynb`
- **Задача**: Аппроксимация набора точек полиномом степени 4 с помощью генетического алгоритма
- **Особенности**:
  - Индивид: вектор коэффициентов [c₀, c₁, c₂, c₃, c₄]
  - Fitness: сумма квадратов ошибок (SSE)
  - Селекция: сортировка по fitness + элитизм
  - Кроссовер и мутация параллелизированы
- **Визуализация**: Графики входных данных, результат аппроксимации и динамика сходимости

### 3. **MatMul** — Умножение матриц
- **Файл**: `HPC_MatMul_6131.ipynb`
- **Задача**: Параллельное умножение двух матриц размером 100×100 до 2000×2000
- **Особенности**:
  - CPU: три вложенных цикла, O(n³)
  - GPU (CUDA + Numba): распараллеливание внешних двух циклов
  - Каждый поток вычисляет один элемент результирующей матрицы
  - Ускорение GPU: 15000× на матрицах 1600×1600
- **Результаты**: Таблица с временем выполнения и коэффициентом ускорения

### 4. **Bilateral** — Bilateral-фильтр
- **Файл**: `HPC_Bilateral_6131.ipynb`
- **Задача**: 9-точечная фильтрация полутонового BMP-изображения
- **Особенности**:
  - CPU-реализация классического bilateral-фильтра
  - GPU-реализация (CUDA/Numba)
  - Граничные условия: nearest pixel (clamp)
  - Параметры: σ_d (пространственная), σ_r (range-составляющая)
- **Входные данные**: `input1.bmp`
- **Выходные данные**: `output_cpu1.bmp`, `output_gpu1.bmp`

### 5. **PI Calc** — Вычисление числа π
- **Файл**: `HPC_PICalc_6131.ipynb`
- **Задача**: Вычисление π методом Монте-Карло
- **Особенности**:
  - Параллельное вычисление случайных точек
  - Проверка попадания в единичный круг
  - Статистическое вычисление π ≈ 4 × (точки в круге) / (всего точек)
- **Достоинства распараллеливания**: Независимость вычислений для каждой точки
